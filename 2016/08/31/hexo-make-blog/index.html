第一步 搭建Hexo环境

Hexo官方参考文档

安装

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

  • Node.js
  • Git

这两项安装方式自行搜索，网上一堆的方法

安装 Hexo：

建站：

创建一个文件夹：“blog”，用户存放Hexo的配置文件，然后进入blog敲击命令初始化

基本完成，之前创建的blog目录就是你博客的根目录，所有hexo环境及配置文件都在里面，接下来就是博客生成三部曲：

  1. 清除命令 hexo clean
  2. 生成静态页面命令 hexo g
  3. 启动服务 hexo s

浏览器输入 http://localhost:4000 你将看到属于你的博客啦:

当然，现在还仅仅是在本地运行着，接下来将其放入Github上!

在github上进入你上面创建的项目然后：

第五步 写博客吧！

到了这部分，你完全可以参考官方的套路来创建博文，也可自己按照自己的方式写

一些常用命令：

hexo new”postName” 新建文章
hexo new page”pageName” 新建页面
hexo g 生成静态页面至public目录
hexo s 开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy 将.deploy目录部署到GitHub
hexo help 查看帮助
hexo version 查看Hexo的版本
